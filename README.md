
# Tileset Formatter

Tool I made to reformat tilesets to be used for a [gamemaker autotiling library](https://github.com/iAmMortos/autotile)

## Usage

Run `main.py`, when prompted, input an image formatted like so:

![proper tile format](./image/tilesettest.png)

An image will be saved in the same directory, formatted to be easy to import.

## Running

Pillow is the main dependancy.
